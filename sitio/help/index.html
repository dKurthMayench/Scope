Manual de usuario de Scope

¿Estás atascado en alguna página? ¿No sabes qué significa "Contraseña incorrecta"? En ese caso, estás en el sitio indicado. Aquí encontrarás información detallada sobre cómo funciona Scope. Simplemente haz clic en la sección de la barra lateral que quieras, o baja con la rueda del raón hasta encuentrar lo que buscas.

Inicio de sesión

Esta es la página que se muestra por defecto al usuario. Aquí podrá iniciar sesión para empezar a utilizar la aplicación.

Usuario

Si tienes una cuenta de Scope, deberás introducir el nombre de usuario (alias) que elegiste al crearla, y darle al botón para que se muestre el campo para introducir la contraseña.

Contraseña

En este campo deberás introducir la contraseña que elegiste al crear la cuenta. Luego dale a la tecla "enter" o haz clic en el botón para finalizar el inicio de sesión.

Si has introducido un nombre de usuario erróneo en el campo anterior, puedes volver atrás con la tecla "esc" o haciendo clic en el botón .

Mensajes de error

  • Este campo es obligatorio:

    • Aparecerá cuando dejes en blanco un campo obligatorio (usuario o contraseña).

  • El usuario no existe. ¿Crear cuenta?:

    • Aparecerá cuando introduzcas un nombre de usuario que no esté registrado. Comprueba que lo hayas escrito correctamente, o crea una cuenta si aún no lo has hecho.

  • La contraseña no es correcta:

    • Aparecerá cuando la contraseña introducida no coincida con la contraseña establecida en la creación de usuario. Comprueba que la hayas escrito correctamente, y asegurate de que estás intentando iniciar sesión con el nombre de usuario correcto.

Registro de una cuenta

En esta página se podrá crear una cuenta, para luego poder iniciar sesión con ella.

Información

    • En este campo deberás especificar el nombre de usuario que quieras.

    • En este campo deberás introducir tu correo electrónico. Asegurate de escribirlo correctamente ya que deberás verificarlo más adelante.

    • En estos campos deberás introducir una contraseña segura (mínimo 8 caracteres, 1 mayúscula, 1 minúscula y 1 número), y escribirla otra vez para evitar errores.

    • Para poder crear una cuenta deberás aceptar la política de privacidad.

Para seguir con el registro haz clic en el botón .

Mensajes de error

  • Este campo es obligatorio:

    • Aparecerá cuando dejes en blanco un campo obligatorio.

  • El usuario ya existe. ¿Iniciar Sesión?:

    • Aparecerá cuando introduzcas un nombre de usuario que ya está registrado. Inicia sesión si ese usuario es tuyo, o escoge otro nombre de usuario.

  • Introduce máximo 16 caracteres:

    • Aparecerá cuando introduzcas un nombre de usuario con una longitud superior a 16 caracteres.

  • Sólo se pueden utilizar letras, números, "-", "_" o ".":

    • Aparecerá cuando introduzcas un nombre de usuario con símbolos especiales no permitidos.

  • Este correo ya está asociado a una cuenta:

    • Aparecerá cuando el correo introducido ya esté asociado a un usuario. Comprueba que lo hayas escrito correctamente, o prueba a iniciar sesión si estás seguro de haber creado una cuenta.

  • Introduce un correo electrónico válido:

    • Aparecerá cuando el correo introducido no tenga un formato de correo electrónico. Asegurate de que lo hayas escrito correctamente.

  • La contraseña debe tener al menos 8 caracteres, máximo 32, y al menos 1 minúscula, 1 mayúscula y 1 número:

    • Aparecerá cuando la contraseña introducida no cumpla con los requisitos especificados en el mensaje de error.

  • Las contraseñas no coinciden:

    • Aparecerá cuando la contraseña introducida en el primer campo no coincida con la introducida en el segundo.

  • Tienes que aceptar la política de privacidad para poder seguir con la creación decuenta:

    • Para crear una cuenta deberás marcar la casilla para aceptar la política de privacidad.

Inicio

Tras iniciar sesión, el usuario será redirigido a esta página.

Cabecera

La cabecera estará presente en todas las páginas salvo la de inicio de sesión, registro y política de privacidad.

Constará de un botón para volver a la página de inicio, una barra de búsqueda, y una sección la cual llevará a la página de perfil en caso de hacer clic en ella.

Navegación

Constará de un botón para volver a la página de inicio, una barra de búsqueda, y una sección la cual llevará a la página de perfil en caso de hacer clic en ella.
